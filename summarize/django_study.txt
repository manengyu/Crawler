安装：
pip install Django
或https://www.djangoproject.com/download/
开始：
django-admin.py startproject name
cd name
python manage.py runserver 127.0.0.1:8000(IP端口可省略)
在项目下创建APP
python manage.py startapp name
models.py文件里继承models.Model创建字段
通过python manage.py makemigrations
python manage.py migrate创建表和数据库
Django框架功能全面使用时需学习段时间，集成了ORM


Flask使用简单方便，但用的技术相对另Django,Tornado较深，常结合ORM SQLAlchemy
http://docs.jinkan.org/docs/flask/quickstart.html#redirects-and-errors
